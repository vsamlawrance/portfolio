.NET Web Application for Employee Data Visualization

This ASP.NET web application was developed using the .NET Framework with ADO.NET for efficient database connectivity and data management. The system retrieves employee information from SQL Server through stored procedures, processes their details—including CTC and increment percentages for employees below the Managing Director—and securely stores them in the database.

Data visualization is seamlessly integrated using a Tableau dashboard connected to the application for real-time insights. The project was built in Visual Studio using C# as the core programming language, ensuring modular and maintainable code structure.

Security measures include password hashing and Base64 string encoding, along with session management for user authentication and secure data handling. Additionally, the application features dynamic form entries, allowing users to select multiple years and input corresponding performance ratings, providing flexibility and scalability in employee data analysis.

Fake Profile Detection using Blockchain

This Flask-based web application integrates machine learning and blockchain to detect and record fake social media profiles securely. The system extracts several key features such as profile picture availability, username structure, biography length, external URL presence, and engagement metrics including followers, following, and post counts.

Using ensemble machine learning models like Gradient Boosting, Random Forest, and Support Vector Machine, the system classifies whether a profile is genuine or fake with high accuracy. Each verification result is then stored on a blockchain ledger, ensuring transparency and immutability of data.

The project includes a clean Flask-based interface for users to submit profiles and view detection results, backed by a SQL database for record storage and model updates. This integration of AI with blockchain enhances online authenticity verification, providing a secure, auditable, and tamper-proof solution for identifying fake profiles.

Digital Document Viewer Web Application

The application provides a centralized interface optimized for large TV screens, ensuring that operators and supervisors can quickly access and display important documents during production or maintenance.

Dynamic search and filter options were implemented to improve navigation across hundreds of documents.

The backend logic was developed using C# with ADO.NET for database connectivity and CRUD operations, while data was securely stored and retrieved through stored procedures. The design prioritizes responsiveness, user accessibility, and seamless integration with the company’s internal network. This project showcases my ability to combine backend logic, database management, and user-centered design in a practical, industry-oriented solution.

Inventory Management System using DBMS

This project focuses on designing and implementing an Inventory Management System using fundamental database management concepts. It enables efficient handling of product details, supplier information, stock levels, and transaction records to maintain real-time inventory tracking.

Developed using SQL as the core backend, the system utilizes relational database design principles including primary and foreign key constraints, normalization, and stored procedures to ensure data accuracy and consistency. It supports all CRUD operations such as adding, updating, deleting, and retrieving records through optimized SQL queries.

The application also includes trigger-based alerts for low stock levels and automatic logging of stock movement for better auditability. This project demonstrates my understanding of relational schema design, data modeling, query optimization, and database-driven application logic for managing large-scale inventory systems effectively.
